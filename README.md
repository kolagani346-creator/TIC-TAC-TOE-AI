# TIC-TAC-TOE-AI
    Tic Tac Toe AI – Intermediate Level Project.Developed an interactive Tic Tac Toe AI game using Python and Kivy with a user-friendly GUI and single-player gameplay. Implemented features such as automatic turn handling, winner detection, clear game board display, and automatic restart after game completion. Through this project, I improved skills in Python programming, GUI development, game logic, debugging, event handling, and problem solving. Future enhancements include implementing the Minimax algorithm, adding difficulty levels, multiplayer mode, score tracking, and UI improvements for a smarter and more engaging experience.
