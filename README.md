# TIC-TAC-TOE-AI
     Tic Tac Toe AI is an intermediate-level project developed using Python and Kivy that allows a user to play against an AI opponent in single-player mode. The project includes features such as graphical user interface design, game board management, AI-based move selection, automatic turn switching, winner detection, and automatic game restart after completion. During the development process, concepts such as Python programming, Kivy GUI development, event handling, debugging, game logic, problem solving, and basic artificial intelligence implementation were applied and improved. This project helped strengthen practical software development skills and understanding of creating interactive applications.
